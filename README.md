# ProjectMyCV 🚀

A clean, digital curriculum vitae built from scratch using semantic HTML5 and custom CSS3 layouts.

## 🌐 Live Demo
You can view the live, working deployment of this CV directly in your browser here:
👉 **[https://zaimevelenn.github.io/ProjectMyCV/](https://zaimevelenn.github.io/ProjectMyCV/)**

---

## 🛠️ Built With
* **HTML5:** Structured layout using semantic tags and Open Graph metadata setup for clean link shares.
* **CSS3:** Managed with global custom property variables (`:root`), absolute print sizing presets (`210mm` x `300mm` A4 proportions), and flexible box structures (`display: flex`).

---

## 📂 Project Structure
* `index.html` - The main codebase layout holding portfolio data.
* `style.css` - Custom style sheets managing grid positioning and color palettes.
* `123.jpg` - Embedded profile image asset.
* `HTML.png` - Custom browser shortcut favicon.
